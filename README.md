# Anki PTSI Collaboration
## Pourquoi un repo ?
J'avais déjà commencé à créer des packs Anki pour mon utilisation personnelle. Je me suis dit pourquoi pas les partager à qui veut et en a besoin. 
De plus, créer un repo permet à n'importe qui de participer à ce projet. C'est une invitation ! Si vous êtes en PTSI et que vous souhaitez ajouter vos cartes/decks pour rendre ce repo encore plus merveilleux !
